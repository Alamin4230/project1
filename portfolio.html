Projects

Smart Irrigation System

This project focuses on developing a smart irrigation system for agricultural fields, designed to optimize water usage. The system operates by automatically activating irrigation when the moisture level in the soil drops below a predefined threshold, ensuring that crops receive water only when necessary. The system is equipped with various sensors, including a raindrop sensor that detects rainfall, a soil moisture sensor that measures the water content in the soil, a PIR (Passive Infrared) sensor that monitors movement, and sensors for temperature and humidity to track environmental conditions. By combining these sensors, the system can efficiently manage water resources, conserve water, and ensure that crops are watered according to their specific needs. This project highlights the potential of IoT (Internet of Things) technologies in transforming traditional farming practices, making them more sustainable, efficient, and less reliant on human intervention.

A Village Playground Using Glut

This project is a computer graphics application designed to visualize a village environment. Using OpenGL (GLUT), the application presents a 3D model of a village playground, which allows users to explore the area and interact with various objects and surroundings in a simulated environment. The primary goal of the project is to demonstrate the power of computer graphics in creating realistic and immersive simulations. The project uses OpenGL to render the 3D graphics, enabling the visualization of objects, landscapes, and architectural structures with high precision. This application showcases the potential of computer graphics in fields like urban planning, education, and entertainment, providing a dynamic way to visualize and interact with conceptual designs before actual implementation.
